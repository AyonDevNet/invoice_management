Database Layer (SQLite/PostgreSQL)

Invoice Table Structure

Field Name Data Type Description
id INTEGER Primary Key, Auto-increment
serial_number VARCHAR(50) Unique invoice serial number
device_name VARCHAR(200) Name of device/product
customer_name VARCHAR(200) Customer information
invoice_date DATE Date of invoice
amount DECIMAL(10,2) Invoice amount
file_path VARCHAR(500) Path to uploaded file
notes TEXT Additional notes
created_at TIMESTAMP Record creation time
